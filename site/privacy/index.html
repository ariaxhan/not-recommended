The short version

We don’t want your data.

Not Recommended has no accounts, analytics, ads, tracking, or telemetry. Your extension data stays in your browser.

Last updated: July 16, 2026

01

What the extension stores

The extension stores the following in chrome.storage.local, inside the Chrome profile where it is installed:

  • display settings;
  • question threads;
  • saved YouTube video titles, IDs, URLs, and save times;
  • your most recent intention sentence, if you enter one; and
  • the current Wikipedia detour set and draw date.

This information is not sent to us. There is no account or remote database.

02

Where the extension connects

The content script runs on youtube.com so it can replace and simplify YouTube surfaces. Your use of YouTube remains subject to YouTube’s own terms and privacy practices.

The background worker makes fixed, credential-free requests to the English Wikipedia API when it needs a daily detour set or when you request a new one. It does not send your saved videos, questions, settings, intentions, YouTube activity, or stored identifiers to Wikipedia.

The worker accepts only its fixed Wikipedia request. It cannot be turned into a general network proxy.

03

What this website does

This website is static. It has no forms, accounts, analytics, advertising, tracking scripts, or cookies set by Not Recommended.

Cloudflare delivers and protects the website. Like other hosting providers, Cloudflare may process technical request information such as IP addresses, routing data, and security signals. That processing is governed by Cloudflare’s privacy policy.

04

How to remove your data

You can delete individual question threads and saved videos from the extension’s intentional home screen. Removing the extension through Chrome removes its local extension storage.

05

Changes and questions

If this policy changes, the date at the top of this page will change too. For privacy questions, use the support contact shown on the extension’s Chrome Web Store listing.
